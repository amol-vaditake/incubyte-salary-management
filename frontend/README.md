# Frontend

React + TypeScript, built with Vite. State via Zustand, styling via Tailwind CSS
and shadcn/ui, routing via React Router.

See the repo root [README.md](../README.md) for the full project overview.

## Local development

```bash
npm install
cp .env.example .env   # set VITE_API_URL to the backend's address
npm run dev
```

## Build

```bash
npm run build
```
